Click the bell icon in the top-right toolbar to open the changelog drawer.
This is a minimal demo page that simulates a BRMS-like application shell. The changelog widget lives in the topbar and can be embedded in any app with a single <script> tag. <script src="changelog-widget.js"></script> <changelog-widget theme="brms"></changelog-widget> The widget self-loads the SDS theme CSS and loader if they are not already on the page, so it works in any HTML environment. When SDS is already loaded (as in BRMS), the widget reuses the existing instance. The widget currently uses dummy data. Once the Nebula enrichment API is ready, pass the endpoint via the api-url attribute and the widget will fetch live release notes on open. <changelog-widget api-url="https://nebula.scordigital.solutions/api/v1/releases/latest"></changelog-widget>
